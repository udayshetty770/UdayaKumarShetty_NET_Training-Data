
# Week 4: Full Stack Project – Student Management System

## Objective
Consolidate all concepts learned in previous weeks by building a complete **Full Stack Student Management System**.  
This project integrates **ASP.NET Core Web API** (backend), **SQL Server** (database), and **ReactJS** (frontend) to create a fully functional CRUD application.

---

## Project Overview
The Student Management System allows users to:
- Add, edit, delete, and view student records.  
- Manage data through an ASP.NET Core API connected to SQL Server.  
- Interact via a responsive ReactJS-based user interface.  
- Demonstrate complete full-stack integration from database to frontend.

---

## Architecture
**Tech Stack:**  
- **Frontend:** ReactJS, Bootstrap  
- **Backend:** ASP.NET Core Web API (C#)  
- **Database:** Microsoft SQL Server  
- **ORM:** Entity Framework Core (Database First)  
- **Version Control:** Git and GitHub  

**Layers Implemented:**  
1. **Database Layer:** SQL tables and stored procedures.  
2. **Backend Layer:** ASP.NET Core Web API for CRUD operations.  
3. **Frontend Layer:** ReactJS for user interaction and API consumption.
